[Hiring] Python Developer Internship @Центр развития карьеры Skillbox

4d ago - Центр развития карьеры Skillbox is hiring a remote Python Developer Internship. 💸 Salary: unpaid internship 📍Location: Russia

Role Description

Компания в поиске Python-разработчика на практику.

  • Разработка и обучение нейросетей для прикладных задач;
  • Создание чат-ботов для различных платформ (Telegram, WhatsApp и другие);
  • Участие в создании прикладных программ под реальные бизнес-задачи;
  • Проектная работа в команде профессионалов;
  • Код-ревью и участие в процессе улучшения качества кода.

Qualifications

  • Базовые знания Python (умение работать с библиотеками для машинного обучения будет плюсом);
  • Опыт работы с API желателен (RESTful API, Webhooks);
  • Знакомство с фреймворками Flask или FastAPI;
  • Умение работать с базами данных (SQL, NoSQL).

Requirements

  • Git для контроля версий (Git Flow будет плюсом);
  • Работа с библиотеками для разработки чат-ботов (например, aiogram, pyTelegramBotAPI);
  • Основы машинного обучения (scikit-learn, TensorFlow или PyTorch);
  • Умение играть в команде и желание учиться новому.

Benefits

  • Неоплачиваемая практика;
  • Длительность: 4 месяца;
  • Формат: удаленно;
  • График обсуждается, занятость 4 часа в день;
  • Сертификат о прохождении практики;
  • Рекомендательное письмо от директора агентства с картой компетенций по результатам практики;
  • Формируем сильное портфолио из предоставленных агентством проектов;
  • Возможность получить оффер;
  • Возможность совместного написания экспертных статей в электронные СМИ.

Company Description

Digital-агентство «S1» – AI-агентство онлайн-маркетинга с акцентом на нейросети.

  • У нас классное руководство и лояльное отношение к сотрудникам;
  • Акцентируем внимание на цифрах и KPI для клиента, эффективности создаваемых продуктов.
